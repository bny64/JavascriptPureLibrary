# 업무 관리 시스템

폐쇄망 환경에서 사용 가능한 파일 기반 업무 관리 시스템입니다.

## 특징

- ✅ Node.js 기본 모듈만 사용 (외부 패키지 불필요)
- ✅ 파일 기반 데이터 저장 (tasks.json, categories.json)
- ✅ 대분류, 중분류 트리 구조 지원
- ✅ 캘린더 기반 날짜 관리
- ✅ 업무 추가, 수정, 삭제 기능
- ✅ 진행 상태 관리 (대기, 진행중, 완료, 보류)
- ✅ 전체 업무 목록 (페이지네이션)
- ✅ 분류 관리 시스템
- ✅ 반응형 녹색 테마 디자인

## 시스템 요구사항

- Node.js (v12 이상)

## 설치 및 실행

1. 모든 파일을 동일한 디렉토리에 위치시킵니다:
   - server.js
   - index.html
   - styles.css
   - app.js

2. 서버 실행:
   ```bash
   node server.js
   ```

3. 웹 브라우저에서 접속:
   ```
   http://localhost:3000
   ```

## 파일 구조

```
project/
├── server.js          # Node.js 서버 (API 제공)
├── index.html         # 메인 HTML
├── styles.css         # 스타일시트
├── app.js             # 클라이언트 JavaScript
├── tasks.json         # 업무 데이터 (자동 생성)
└── categories.json    # 분류 데이터 (자동 생성)
```

## 화면 구성

### 상단 헤더
- 분류 관리 버튼
- 새 업무 추가 버튼

### 왼쪽 (75% - 캘린더 영역)
- 월별 캘린더 표시
- 업무가 있는 날짜에 상태별 점(dot) 표시
- 날짜 클릭으로 해당 날짜 업무 확인

### 오른쪽 상단 (15% - 선택된 날짜의 업무)
- 선택한 날짜의 업무 목록 표시
- 업무별 수정/삭제 버튼 제공

### 오른쪽 하단 (10% - 전체 업무 목록)
- 모든 업무를 날짜순으로 표시
- 페이지네이션 (5개씩)
- 업무 클릭 시 해당 날짜로 이동

## 주요 기능

### 1. 분류 관리
- "분류 관리" 버튼 클릭
- 대분류만 또는 대분류 + 중분류 조합으로 등록
- 트리 구조로 계층적 표시
- 수정/삭제 기능

### 2. 업무 추가
- "새 업무 추가" 버튼 클릭
- 등록된 분류에서 선택
- 업무명, 시작/종료 날짜, 진행상태 설정
- 선택적으로 상세 설명 추가

### 3. 업무 수정
- 업무 항목의 "수정" 버튼 클릭
- 정보 수정 후 저장

### 4. 업무 삭제
- 업무 항목의 "삭제" 버튼 클릭
- 확인 후 삭제

### 5. 진행 상태
- 대기 (노란색)
- 진행중 (파란색)
- 완료 (초록색)
- 보류 (회색)

### 6. 날짜 관리
- 캘린더에서 날짜 선택
- 업무 기간 동안 모든 날짜에 표시
- 오늘 날짜는 자동으로 강조 표시

### 7. 전체 업무 목록
- 날짜순으로 정렬된 모든 업무
- 5개씩 페이지네이션
- 업무 클릭 시 해당 날짜로 이동

## 데이터 저장

모든 데이터는 JSON 파일에 저장됩니다:
- `tasks.json`: 업무 데이터
- `categories.json`: 분류 데이터

서버를 재시작해도 데이터가 유지됩니다.

### 데이터 백업
두 JSON 파일을 복사하여 백업할 수 있습니다.

### 데이터 복원
백업한 JSON 파일을 서버 디렉토리에 복사합니다.

## 포트 변경

기본 포트는 3000입니다. 변경하려면 `server.js`의 `PORT` 변수를 수정하세요:

```javascript
const PORT = 3000; // 원하는 포트 번호로 변경
```

## 주의사항

1. 폐쇄망 환경에서 사용하도록 설계되었습니다
2. 외부 CDN이나 인터넷 연결이 필요하지 않습니다
3. 다중 사용자 동시 접속 시 데이터 충돌 가능성이 있습니다
4. 중요한 데이터는 정기적으로 백업하세요
5. 분류를 먼저 등록한 후 업무를 추가하세요

## 문제 해결

### 서버가 시작되지 않는 경우
- Node.js가 설치되어 있는지 확인
- 포트가 이미 사용 중인지 확인

### 데이터가 저장되지 않는 경우
- 파일 쓰기 권한 확인
- JSON 파일 손상 여부 확인

### 화면이 제대로 표시되지 않는 경우
- 브라우저 캐시 삭제
- 모든 파일이 같은 디렉토리에 있는지 확인

### 모달이 잘리는 경우
- 브라우저 화면을 최대화하거나 확대/축소 비율 조정
- 모달은 스크롤 가능하도록 설정되어 있습니다
